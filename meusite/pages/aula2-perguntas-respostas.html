Perguntas & respostas

Aqui vamos ter perguntas e respostas da aula de hoje.

Qual a função dos comentários?

R.: Os comentário ajudam a nortear apontamentos realizados pelo desenvolvedor e equipe, este são nada mais que notas para explicar o que está sendo desenvolvido.

Qual a função da tag charset="UTF-8"?

R.: A tag charset define o padrão de códificação para os caracteres. Este utilizado, consta parte dos nossos caracteres utilizados na linguagem português (Brasil).

Qual é a função do HTML e CSS?

R.: A função do HTML é definir a marcação para a página web, ou seja, com ele podemos criar textos, formata-los através das tags para que assim seja exibidos em navegadores. Ao agregarmos o CSS podemos estilizar nossa página HTML, assim adicionamos cores, novos designs e posteriormente adicionamos o Javascript.

Qual o nome padrão da página principal?

R.: O nome padrão da página principal é index ou default com a extensão .html para assim definir qual o tipo de arquivo.

Qual a função da tag BR?

A tag br é utilizada para quebra de linha.
